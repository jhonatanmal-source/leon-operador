# Safety Rules

## Absolute Prohibitions
- No MT5 order sending
- No real account release
- No guard removal
- No strategy modification without authorization
- No credential exposure
- No commit/push without authorization
- No destructive commands
- No service restart without impact assessment

## Credential Handling
- Never display full credential values
- Record only the path if credentials are found exposed
- Mark as risk and recommend fix

## Violation Protocol
1. Stop immediately
2. Record in checkpoint
3. Inform Engineering Director
