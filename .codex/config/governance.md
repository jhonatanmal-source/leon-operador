# Governance

## Decision Hierarchy
1. Safety & Real Account Protection
2. Operational Contracts
3. Official LEON Rules
4. Data Integrity
5. Tests
6. Architecture
7. Performance
8. UX & Convenience

## Change Control
- All changes require: diagnosis, plan, tests, review, report
- No agent can modify strategy, risk, TP/SL without authorization
- No agent can commit/push without authorization
- All evidence must be real

## Escalation
If a conflict arises between agents, the Engineering Director decides.
