# Padrões do Projeto

## Agentes OpenCode
- Formato: Markdown com frontmatter YAML
- Local: `.opencode/agents/`
- Nomenclatura: `leon-<funcao>.md`

## Skills
- Formato: `SKILL.md` com frontmatter YAML
- Local: `.opencode/skills/<nome>/`
- Nomenclatura: `leon-<descricao-com-hifens>`

## Comandos
- Formato: Markdown com frontmatter YAML
- Local: `.opencode/commands/`
- Nomenclatura: `leon-<comando>.md`

## Arquivos de Missão
- Checkpoint: `tarefas/checkpoint_equipe_leon.json` (JSON) + `checkpoint_equipe_leon.md` (legível)
- Diagnóstico: `tarefas/diagnostico_*.md`
- Relatório: `tarefas/relatorio_final_*.md`
- Memória: `tarefas/memoria_engenharia/*.md`
