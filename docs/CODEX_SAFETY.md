# Codex Safety

## Absolute Prohibitions
- No MT5 order sending
- No real account release
- No guard removal
- No strategy modification without authorization
- No credential exposure
- No commit/push without authorization

## Protected Contracts
1. Real account always blocked
2. Single executor
3. PRE_OPERATION required
4. Technical RR/SL/TP only
5. Max 3 trades per day
6. News shield active
7. Structural confirmation required

## Violation Protocol
1. Stop immediately
2. Record in checkpoint
3. Inform Engineering Director
