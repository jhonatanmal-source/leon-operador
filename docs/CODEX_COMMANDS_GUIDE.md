# Codex Commands Guide

## Available Commands
Commands are defined in `.codex/commands/` and `.opencode/commands/`.

| Command | Purpose |
|---------|---------|
| `/leon-missao` | Start a mission |
| `/leon-corrigir` | Fix a bug |
| `/leon-auditar` | Read-only audit |
| `/leon-testar` | Run tests |
| `/leon-revisar` | Review changes |
| `/leon-refatorar` | Refactor code |
| `/leon-otimizar` | Optimize performance |
| `/leon-vps` | VPS diagnostics |
| `/leon-seguranca` | Security check |
| `/leon-observabilidade` | Observability review |
| `/leon-release` | Prepare release |
| `/leon-status` | Mission status |
| `/leon-retomar` | Resume mission |
| `/leon-relatorio` | Generate report |
