# Codex Skills Guide

## Skill Structure
Each skill in `.codex/skills/<id>-<name>/SKILL.md` contains:
- Name and identifier
- Objective
- When to activate
- Responsibilities
- Rules
- Dependencies
- Procedure

## Skill List
See `.codex/README.md` for the complete list of 25 skills.

## Best Practices
1. Always load the skill's full content before starting
2. Follow the procedure step by step
3. Respect the rules and protected contracts
4. Generate evidence for all steps
