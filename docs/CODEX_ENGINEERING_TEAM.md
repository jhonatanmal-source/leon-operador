# Codex Engineering Team

## Overview
The Codex Engineering Team is a professional skill library for the LEON XAU ELITE AI project, providing 25 specialized skills covering all engineering domains.

## Team Structure
- **Engineering Director**: Mission coordination and oversight
- **Senior Software Engineer**: Implementation and bug fixes
- **Software Architect**: System architecture
- **24 Specialized Skills**: Covering trading systems, SMC, risk, MT5, Telegram, dashboard, AI, testing, security, DevOps, and more

## How to Use
1. Start a mission: Select the appropriate skill(s)
2. Follow the skill's procedure
3. Use templates for reports
4. Run validators before completion

## Location
`.codex/` directory in project root.
